
#  MoneyPlant AI 

TO assess the functionality of ERC20 and NFT's





## Installation

Install with npm

```bash
  cd frontend
  yarn ```
    

Go to the project directory

```bash
  cd wallet-api
```

Install dependencies

```bash
  yarn 
```

Start the server

```bash
yarn dev
```
